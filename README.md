# AgriResolve-AI
An agentic, safety-first computer vision system for crop leaf assessment. Uses multi-agent reasoning, evidence quality checks, and deterministic arbitration to avoid overdiagnosis. Supports multilingual queries, voice input, and grounded government subsidy guidance without unsafe medical claims.
